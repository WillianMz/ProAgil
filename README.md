# Proagil Eventos

Aplicação desenvolvida no curso de Dotnet Core com Angular

## Requisitos

``` bash
Dotnet Core 2.2;
Angular;
NodeJs
```

## Instalação

``` bash
git clone https://github.com/WillianMz/ProAgil.git
cd Proagil-App
npm install
```

## Executar a aplicação (Ambiente de Desenvolvimento)

Executar a API
```cs
cd Proagil
cd Proagil.API
dotnet watch run
```

Executar aplicacao Angular
```js
cd Proagil
cd Proagil-App
ng serve
```
